1. create folder
2. initilize git environment:
   " git init  " 
3. add to git staging area:
   " git add .  "
4. checking status:
   "  git status   "
5. remove files from staging area:
   "  git rm --cached -r . "
6. to commit files with message:
   "  git commit -m "Initial submit"   "

7. check commiting history:
   "  git log  "
8. to create new branch:
   "  git branch namename   "
9. check what branch you have:
   "   git branch  "
10. switch to different branch:
   "  git checkout branch-name   "
11. mergit branch to master:
   " git merge branch-name   "
12. push to remote repo:
   " git remote add origin https:...(copy from your github repo page  "
   " git push -u origin master   "
